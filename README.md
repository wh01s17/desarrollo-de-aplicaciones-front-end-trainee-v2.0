# Desarrollo de Aplicaciones Front-End Trainee V2.0

Repositorio asociado al curso **Desarrollo de Aplicaciones Front-End Trainee V2.0**, impartido por **Sustantiva** en el marco del programa **Talento Digital para Chile - SENCE**.

## 📌 Descripción

El curso está diseñado para entregar las habilidades necesarias para construir y mantener aplicaciones o sitios web utilizando el stack de tecnologías fundamentales para el desarrollo front-end: **HTML**, **CSS** y **JavaScript**. Las y los participantes desarrollarán competencias para crear componentes visuales, implementar interfaces interactivas con el framework **Vue.js**, aplicar buenas prácticas de la industria y crear soluciones digitales que respondan a problemáticas reales de las organizaciones.

## 🎯 Competencia del Plan Formativo

**Desarrollar aplicaciones web desde el lado del cliente**, que den solución a las necesidades de la organización, y aplicando las buenas prácticas de la industria para obtener un producto con niveles de calidad requeridos.

## 🧑‍💻 Campo Laboral

Los egresados podrán desempeñarse en organizaciones públicas y privadas de distintas industrias (retail, banca, salud, minería, manufactura, servicios, TI), en roles de **desarrollador/a front-end trainee o junior**, integrándose a equipos de proyectos de software.

## 📝 Requisitos de Ingreso

- Licencia de Enseñanza Media completa.
- Manejo computacional a nivel usuario.
- Aprobación de test de selección (raciocinio lógico, comprensión verbal y aptitud).

## 📚 Módulos del Curso (Total: 438 horas)

1. **Orientación al Perfil y Metodología del Curso** (18h)
2. **Fundamentos de Desarrollo Front-End** (72h)
3. **Desarrollo de la Interfaz de Usuario Web** (56h)
4. **Fundamentos de Programación en JavaScript** (60h)
5. **Programación Avanzada en JavaScript (V2)** (48h)
6. **Desarrollo de Interfaces Interactivas con Framework Vue** (72h)
7. **Desarrollo de Aplicaciones Front-End con Framework Vue** (76h)
8. **Desarrollo de Portafolio de un Producto Digital** (18h)
9. **Desarrollo de Empleabilidad en la Industria Digital** (18h)

---

## 🚀 Tecnologías Clave

- **Lenguajes:** HTML5, CSS3, JavaScript (ES6+)
- **Framework:** Vue.js (Vue Router, Vuex)
- **Preprocesadores:** SASS
- **Framework CSS:** Bootstrap 5
- **Consumo de APIs:** Axios, Fetch, XHR
- **Backend as a Service:** Firebase
- **Testing:** Jest, Mocha + Chai, Cypress
- **Control de Versiones:** Git/GitHub
- **Metodologías:** Bootcamp, trabajo colaborativo, metodologías ágiles

## 🎓 Requisitos de Aprobación

- Contar con al menos un 40% de horas efectivas de conexión en la plataforma (175 hrs).
- Haber completado al menos el 90% de las actividades obligatorias.
- Obtener una evaluación final equivalente a nota 4.0 o superior.

## 👩‍🎓 Perfil de Egreso

Al finalizar, los participantes estarán preparados para:

- Construir interfaces de usuario web responsivas y accesibles.
- Desarrollar aplicaciones front-end interactivas con Vue.js.
- Consumir APIs REST e integrar servicios externos.
- Implementar pruebas unitarias y end-to-end.
- Trabajar en equipos de desarrollo aplicando buenas prácticas y control de versiones.
- Desarrollar un portafolio digital con proyectos reales.
